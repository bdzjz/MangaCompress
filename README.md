0.2版本

界面功能均已实现

WEBP与JPG二选一，PNG选项去除。因为想了想，追求无损收藏的也没必要压缩成PNG的。

分辨率压缩目前采用总像素，分别为1920x1080的4倍，3倍，2倍，并压缩成1/4,1/3,1/2大小。可自行调整。Else逻辑，符合高一档会忽略低一档。

添加质量参数选项。默认75，感觉是够用的，文件大小也可接受。

0.1版本目前仅实现基础功能，所有选项均无用中……

拖拽文件到右侧文本框，点击开始压缩。目前只处理ZIP包。

会在固定路径下生成Temp，TempAfter，TempZIP三个文件夹（写死的路径可自行修改，原来写的压缩包相对路径备用）。第一个为解压临时路径，第二个为图片压缩后路径，第三个为压缩后打包ZIP路径。

完成之后删除前2个文件夹下的临时文件。为了避免误删除，添加BDZJZ.tmp文件作判断，删除使用回收站，最大限度避免意外。
